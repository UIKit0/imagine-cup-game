Gearset Install Instructions

For the latest installation instruction and User Guide, go to: http://www.thecomplot.com/gearsetguide.html

HOW TO USE IT IN YOUR PROJECT

  1. Download Gearset.dll from http://www.thecomplot.com/gearsetdownload.html
  2. Include Gearset.dll as a reference in your XNA 4.0 project.
  3. Copy the included GS.cs file to your project, it provides you with quick access to Gearset's methods and properties while allowing you to completely remove Gearset when your game is ready to be shipped.
  4. Right click on your project -> Properties -> Application -> Target Framework. And make sure it says ".NET Framework 4" instead of ".NET Framework 4 Client Profile".
  5. Right click on your project -> Properties -> Build -> Conditional Compilation Symbols -> append ";USE_GEARSET" to it. When you want to completely remove gearset, simply remove that.
  6. At the beginning of your Game.Initialize method, add this to start Gearset components: 
        public override void Initialize()
    {
        Gearset.GS.Initialize(this);
    }
  7. In your program's Main method (usually located either in Program.cs or your main Game code file) add the [STAThread] attribute: 
        [STAThread]
    static void Main(string[] args)
    {
        ...
    }
  8. Press F5 and run your game!
  9. Aditionally for 3D, or 2D scrolling games your camera matrices must be passed to Gearset in order to render overlays correctly. Inside your Game.Update method add: 
        public override void Update(GameTime gameTime)
    {
        ...
        Gearset.GS.SetMatrices(ref World, ref View, ref Projection, ref Transform2D);
    }

OTHER PLATFORMS

Currently Gearset will only work on Windows, however we provide you with two dummy assemblies (assemblies with no code, only the method signatures) named "Gearset.WP.dll" for Windows Phone and Gearset.Xbox.dll for XBox360 so you don't have to modify your game code to make it compile for other platforms. For this to work you will have to do the following:
  • Remove the reference to Gearset.dll on your WindowsPhone/Xbox project, if any.
  • For a Windows Phone Game: Add a reference to Gearset.WP.dll
  • For an Xbox 360 Game: Add a reference to Gearset.Xbox.dll
